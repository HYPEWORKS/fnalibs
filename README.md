This repo distributes the native libraries for FNA:

  * FNA3D
  * FAudio
  * SDL2
  * Theorafile
  * MoltenVK (for macOS)

## Supported Platforms

The libraries will be built for these platforms/architectures:

  * Windows
    * x86
    * x64
  * Mac OS
    * x64
    * ARM64/AArch64 (Apple Silicon)
  * Linux
    * x64
    * ARM64/AArch64
